标注流程说明

一.界面说明

1.左上角按钮区用于选取标注对象
2.右上方区用于展示标注对象
3.中下方显示50个近似的标注对象

二.左上角按钮区选取标注对象

1.下拉框选取数据集
2.点击“选择标注对象序号范围”按钮，在弹出页面选择要标注的对象序号范围
3.下拉框选取标注结果：自动根据用户名新建选项，也可浏览他人的标注结果
4.（可选）下拉框选区特定ID，主要用于回顾之前的标记，同时展示标注进度

三.中下方进行行人身份标注

1.共显示50个类似用户
2.如果其中一个或多个框内的人和上方的标注对象属于同一个人，请选中对应的选框，会出现打勾以及蓝色边框示意
3.点击中间的按钮“确定，下一个标注”标注下一个
4.如果当前被标注对象，也就是上方图片中序列全部质量差（好的定义是完整的单个人体，差的可能：倒影、双人（抱小孩不算双人）、严重遮挡（气球等)、不全（半身人体）），选择对应的选框进行标注

提示

1.点击小图可以查看大图
2.中下方框内的照片可能较多，可左右拖动切换
3.每个标注都会自动保存
4.请在1920x1080分辨率下使用，推荐使用chrome浏览器
5.问题建议可发邮件至kezaidan@cloudwalk.cn
6.标注准测以“精确”为主，模棱两可的对象就不勾选，质量难以评价的尽量算差
